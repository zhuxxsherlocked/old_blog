List

list_name = [item_1, item_2] 用[]表示
可以是空的 list：empty_list = []
索引 list_name[]
list没有固定长度，在末尾增加item： list.append()

注意index所指的范围。

remove elements from list
n.pop(index) : remove the item of the index and return
del(n[1]) ： 与pop区别在于不返回值
n.remove(item) ： 直接移除item，而不是通过索引

range
举例：
range(6) # => [0,1,2,3,4,5]
range(1,6) # => [1,2,3,4,5]
range(1,6,3) # => [1,4] step是3

Dictionary

与list的区别是用key查询，而不是index。key可以是string或者是number。

词典是可变的，dict_name[new_key] = new_value
删除 del dict_name[key_name]
改变key对应的值 dict_name[key] = new_value
