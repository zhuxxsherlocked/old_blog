数据类型和变量

字符串

以’’或””括起来的任意文本，比如’abc’，”xyz”等等。请注意，’’或””本身只是一种表示方式，不是字符串的一部分，因此，字符串’abc’只有a，b，c这3个字符。如果’本身也是一个字符，那就可以用””括起来，比如”I’m OK”包含的字符是I，’，m，空格，O，K这6个字符。

各种转义字符

如果字符串内部有很多换行，用\n写在一行里不好阅读，为了简化，Python允许用’’’…’’’的格式表示多行内容：

python是动态语言，变量本身类型不固定。
整数做除法或是取余结果仍是整数，其结果永远精确。如果想得准确值，将其中一个整数换作浮点即可。

格式化（还需加强学习）

%运算符是用来格式化字符串的。
在字符串内部，%s表示用字符串替换，%d表示用整数替换，有几个%?占位符，后面就跟几个变量或者值，顺序要对应好。如果只有一个%?，括号可以省略。
常见的占位符有：
%d 整数
%f 浮点数
%s 字符串
%x 十六进制整数

当字符串里%是普通字符时，需要转义，用%%来表示一个%。

List

list是一种有序的集合，可随时添加和删除其中元素。

list_name = [item_1, item_2] 用[]表示
可以是空的 list：empty_list = []

索引
来访问list中每一个位置的元素 list_name[]
从0开始 倒数第一个，第二个：-1 ，-2 以此类推

list没有固定长度，在末尾增加item： list.append(new)
也可把元素插入到指定位置 比如索引号为index list.insert(index,new)

注意index所指的范围。

删除 remove elements from list
n.pop(index) : remove the item of the index and return
del(n[1]) ： 与pop区别在于不返回值
n.remove(item) ： 直接移除item，而不是通过索引

range
举例：
range(6) # => [0,1,2,3,4,5]
range(1,6) # => [1,2,3,4,5]
range(1,6,3) # => [1,4] step是3

替换
直接赋值给对应索引位置

tuple

另一种有序列表叫元组：tuple。tuple和list非常类似，但是tuple一旦初始化就不能修改。更安全。
定义只有一个元素的tuple定义时必须加一个逗号,，来消除歧义：

Python在显示只有1个元素的tuple时，也会加一个逗号,，以免你误解成数学计算意义上的括号。

注意：tuple所谓的“不变”是说，tuple的每个元素，指向永远不变。即指向’a’，就不能改成指向’b’，指向一个list，就不能改成指向其他对象，但指向的这个list本身是可变的！若要创建内容也不变的tuple，则必须保证tuple的每一个元素本身也不能变。

Dictionary

与list的区别是用key查询，而不是index。key可以是string或者是number。key-value。

词典是可变的，dict_name[new_key] = new_value
删除 del dict_name[key_name]
改变key对应的值 dict_name[key] = new_value

set

set和dict类似，也是一组key的集合，但不存储value。由于key不能重复，所以，在set中，没有重复的key。

要创建一个set，需要提供一个list作为输入集合：

注意，传入的参数[1, 2, 3]是一个list，而显示的set([1, 2, 3])只是告诉你这个set内部有1，2，3这3个元素，显示的[]不表示这是一个list。

重复元素在set中自动被过滤。

通过add(key)方法可以添加元素到set中，可以重复添加，但不会有效果。

通过remove(key)方法可以删除元素。

set可以看成数学意义上的无序和无重复元素的集合，因此，两个set可以做数学意义上的交集、并集等操作：

set和dict的唯一区别仅在于没有存储对应的value，但是，set的原理和dict一样，所以，同样不可以放入可变对象，因为无法判断两个可变对象是否相等，也就无法保证set内部“不会有重复元素”。
