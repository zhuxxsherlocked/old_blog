Veins SUMO学习（一）

By Zhuxx

发表于 2015-05-29

文章目录

十天前接触了下SUMO，碰到一堆BUG，一定要选对版本。
感谢CXY。
现在才总结表示有点呵呵。

目的是进行交通仿真。
之前考虑过使用VISSIM，也买了书进行学习，但是VISSIM是商业软件，是要收费的，想画交大地图时，学生版限制在1000m，仿真时间也只有3600s，放弃。
CXY推荐了Veins，开源，免费。但有句话说：免费的是最贵的。 所以，还要下一番功夫。

1.Veins主页 Veins Home
查看Tutorial，需要安装SUMO， OMNeT++， Veins.
注：一定要注意版本
之前装的SUMO msi是0.23.0 教程要求的是0.21.0 执行命令时一直有bug
换回0.21.0 原本没出错的地方，倒出错了。
还有lib库的版本。用0.21.0版本的lib 执行randomTrips.py ，换用0.23.0却能正常运行。但是CXY运行却是好的。
很崩溃！!！有木有！！！
好在，最后跑出了地图。

2.地图导入
参考 https://www.youtube.com/results?search_query=sumo+import+map 第一个和第三个视频
sumo-0.21.0 下 start-command-line.bat
执行以下指令:

cd /d c:\Users\Zhuxx\veins\sumo-0.21.0\sjtu
netconvert --osm-files sjtu.osm -o sjtu.net.xml
polyconvert --net-file sjtu.net.xml --osm-files sjtu.osm --type-file typemap.xml -o sjtu.poly.xml
python c:\Users\Zhuxx\veins\sumo-0.21.0\tools\trip\randomTrips.py -n sjtu.net.xml -e 50 -l
python c:\Users\Zhuxx\veins\sumo-0.21.0\tools\trip\randomTrips.py -n sjtu.net.xml -r sjtu.rou.xml -e 50 -l
sumo-gui sjtu.sumo.cfg

其中牵扯config文件修改，typemap.xml等。
